# Desktop-First Developer Tool Landing Page - Cursor AI

This project is a desktop-only landing page recreated using **pure HTML and CSS**, inspired by the Cursor website.  
The goal of this assignment is to match the **structure, layout, spacing, typography, and visual hierarchy** of the reference design as closely as possible.

# Live Demo

Link: https://sofiya2024.github.io/masterji-cursor-ai-clone/

## Technologies Used

- HTML5
- CSS3

# Sections Implemented

1. Top Navigation Bar  
   - Logo  
   - Navigation links  
   - Primary CTA button  
   - Dark background  

2. Hero Section  
   - Main headline  
   - Description text  
   - CTA button  
   - Large product screenshot  

3. Trusted By / Logos  
   - Horizontal row of company logos  

4. Feature Sections (3 Blocks)  
   - Two-column layout  
   - Alternating text and image alignment  

5. Feature Cards Section  
   - Section title  
   - Grid layout with feature cards  

6. Testimonials  
   - Quote cards  
   - Author name and role  

7. Use Cases / Stories  
   - Cards with image and short description  

8. Changelog / Updates  
   - List-based layout with dates  

9. Team / About  
   - Large image  
   - Short description  
   - CTA button  

10. Final Call To Action  
    - Prominent heading  
    - Single CTA button  

11. Footer  
    - Multi-column links  
    - Company information  

# Fonts and Colors

### Fonts
- Primary font: **Inter**
- Font weights used: Regular, Medium, Bold

### Color Palette
- Dark background: `#14120b`
- Primary text: `#ffffff`
- Secondary text: `#cccccc`

The colors and typography were chosen to closely match the original design.

# Layout Notes

- Desktop-first fixed layout
- Flexbox and CSS Grid used for structure
- Semantic HTML elements (`header`, `section`, `footer`)
- Clean and readable CSS organized by section

# Assets

- Images and icons are selected or approximated to visually resemble the reference website
- All assets are used for educational purposes only
